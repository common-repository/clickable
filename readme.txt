Converts URLs in a post content to clickable links

== Description ==

This simple plugin helps you convert URLs in post content to clickable links. Also, it opens external links in new browser tab automatically.

Just install and activate the plugin, and you're done. No additional settings are required.

For more, visit [https://pluggable.io/plugin/clickable](https://pluggable.io/plugin/clickable)

== Installation ==

1. Upload \`clickable\` to the \`/wp-content/plugins/\` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You're done. Your links in the post contents are now clickable.
